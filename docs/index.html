Projeto James Bond Videogame

Esta é uma lista para organizar minha jornada. Vou jogar todos os jogos já lançados da franquia James Bond. Alguns jogos são acessíveis, modernos e divertidos. Outros são obtusos e obscuros, e serão encarados de forma mais científica.

Regras

  • O objetivo inicial é finalizar todos os jogos.
  • Os jogos devem ser visitados em ordem de lançamento.
  • A lista base foi extraída da wikipédia.
  • Objetivos secundários podem emergir (como completude 100%, dificuldades extras etc.), a ser decidido por jogo.
  • Em caso de jogos com versões diferentes, se forem diferentes o suficiente, todas serão visitadas.
  • Gravar, publicar no youtube, linkar nesta página.
